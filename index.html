AI-Powered Service Finder for Local Communities

LocalFind is an AI-powered local service discovery platform designed to bridge the gap between users and skilled professionals within their communities. The platform enables users to quickly locate and connect with verified local workers such as plumbers, electricians, and other service providers across Tamil Nadu cities. Built with a smart recommendation engine using KMeans Clustering to group users by cost preference and Cosine Similarity to rank and personalize worker suggestions, the system delivers accurate and relevant matches tailored to each user's needs. The platform includes end-to-end user management, worker registration, booking workflows, and administrative oversight, all powered by a Flask REST API and MongoDB backend. LocalFind demonstrates how AI and machine learning can be applied to solve a real-world problem faced by local communities — making skilled help easier to find, faster to access, and smarter to recommend.

Career Match

Career Match is an AI-powered career intelligence platform designed to simplify and strengthen the job search process by intelligently connecting candidates with the right opportunities. The platform enables users to upload their resumes and instantly receive personalized job recommendations based on their skills, experience, and qualifications. By leveraging Natural Language Processing and semantic similarity techniques, the system goes beyond simple keyword matching — understanding the meaning behind a candidate's profile and mapping it to the most suitable job roles. Resume parsing extracts key information such as skills, education, and work experience, while similarity-based matching algorithms rank and recommend the most relevant positions. Web scraping powered by Selenium ensures job listings are current and pulled from real sources, keeping recommendations accurate and up to date. Built with Python, Flask, and SQLite, Career Match demonstrates a practical and meaningful application of AI in solving a real-world challenge that affects millions of job seekers every day.

Stock Portfolio Tracker

Stock Portfolio Tracker is a Python-based command-line application built to help users manage and monitor their personal investment portfolios with ease. The application allows users to add stocks, track real-time portfolio value, view investment summaries, and export data to a file — all through a colorful and intuitive terminal interface built without any external dependencies. Developed using Python's core modules including CSV, OS, and Datetime, the project demonstrates strong fundamentals in data structures, file input/output, error handling, and modular programming design. Robust input validation and exception handling mechanisms ensure data integrity at every step, while user-friendly error messages make the experience smooth and reliable. Stock Portfolio Tracker showcases how clean, well-structured Python code can deliver a practical and efficient solution for real-world financial tracking needs.
